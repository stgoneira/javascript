## Setup Emulador
## [Hola Mundo]() ```bash $ npm install -g cordova $ cordova create holamundo $ cordova platform add browser $ cordova platform add android $ cordova run browser $ cordova run android ```
## [ADB Wifi]() ```javascript # Conecta por USB y asegura que aparezca listado $ adb devices # daemon listening on port XXXX $ adb tcpip 5555 # Buscamos la IP del móvil $ adb shell $ netcfg $ ifconfig # o si no está disponible el cmd anterior # conectamos y ya podemos desconectar el USB $ adb connect <DEVICE_IP_ADDRESS>:5555 ```
## [Desconectar ADB Wifi]() ```bash $ adb kill-server ```

# [Vibración]()
## [Setup]() ```bash $ cordova plugin add cordova-plugin-vibration ```
## [Uso]() ```javascript // navigator.vibrate(time) navigator.vibrate(3000); // vibra x 3 segs // navigator.vibrate([time]) navigator.vibrate([1000, 1000, 3000, 1000, 5000]); // Vibrate for 1 second // Wait for 1 second // Vibrate for 3 seconds // Wait for 1 second // Vibrate for 5 seconds ```

# [Geolocalización]()
## [Setup]() ```bash $ cordova plugin add cordova-plugin-geolocation ```
## Uso ```javascript // usar con navigator.geolocation.... // retorna la ubicación actual del dispositivo getCurrentPosition(success:fn, error:fn, options:obj) // Vigila cambios en la ubicación del dispositivo // retorna watchId watchPosition(success:fn, error:fn, options:obj):String // Para el seguimiento a los cambios de ubicación clearWatch(watchId:String) ```
## Más Info [https://cordova.apache.org/docs/en/latest/reference/cordova-plugin-geolocation/index.html](https://cordova.apache.org/docs/en/latest/reference/cordova-plugin-geolocation/index.html)
