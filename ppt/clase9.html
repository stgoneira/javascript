## [Android Debug Bridge]() ```bash $ adb devices List of devices attached 30044dec0a313100 device ```
## [Introducción Cordova ]() - Permite crear aplicaciones móviles con [HTML, CSS y Javascript]() - Puede utilizar componentes específicos del teléfono a través de [Plugins](): * Cámara * GPS * Contactos * Entre otros - Adobe [Phonegap]() y Apache Cordova
## [Hola Mundo]() ```bash $ npm install -g cordova $ cordova create holamundo $ cordova platform add browser $ cordova platform add android $ cordova run browser $ cordova run android ```
## [Bonus ADB Wifi]() ```javascript # Conecta por USB y asegura que aparezca listado $ adb devices # daemon listening on port XXXX $ adb tcpip 5555 # Buscamos la IP del móvil $ adb shell $ netcfg $ ifconfig # o si no está disponible el cmd anterior # conectamos y ya podemos desconectar el USB $ adb connect <DEVICE_IP_ADDRESS>:5555 ```
## [Desconectar ADB Wifi]() ```bash # Con el cable conectado $ adb -s <DEVICE_IP_ADDRESS>:5555 ```
