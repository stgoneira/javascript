## [Document Object Model (DOM)]() - API para XML - Fue extendida para usar en HTML - Mapea una página HTML a una jerarquía de nodos
## [Ejemplo]() ```html <html> <head> <title>Sample Page</title> </head> <body> <p> Hello World!</p> </body> </html> ```
## [DOM Levels]() - [DOM Level 0](): No es un estándar, usado en Netscape 2 y 4 - [DOM Level 1](): 1998, habilidad para cambiar el documento completo de HTML - [DOM Level 2](): 2001, Introducción de la función "getElementsById", event model y soporte para namespaces - [DOM Level 3](): XPath, keyboard event handling
