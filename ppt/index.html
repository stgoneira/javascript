## Tipos de Datos (ES6) ```javascript let mensaje = "lorem ipsum"; let anoCreacion = 1987; let sinDefinir; let esVerdad = true; console.log(typeof esVerdad); // "boolean" console.log(typeof mensaje); // "string" console.log(typeof anoCreacion); // "number" console.log(typeof sinDefinir); // "undefined" console.log(sinDefinir); // "undefined" console.log(typeof noExiste); // "undefined" console.log(noExiste); // ERROR ```
## Undefined y Null - Null es para Objetos - Null se debe asignar a variables que serán objetos - No es recomendable asignar Undefined ```javascript let car = null; let sinDefinir; console.log(typeof car); // "object" console.log(typeof sinDefinir); // "undefined" console.log(car); // null console.log(sinDefinir); // undefined ```

## [Recomendaciones]() - No usar var - Preferir const sobre let
## [Constantes]() ```javascript // Ej1 const age = 26; age = 36; // TypeError: assignment to a constant // Ej2. const nombre = "Juan"; if(true) { nombre = "Ramón"; // genera error } console.log(nombre); // Ej3. alcance por bloques const nombre = "Juan"; if(true) { const nombre = "Ramón"; // NO genera error } console.log(nombre); ```
## [var y let]() ```javascript var saludo = "holi"; var saludo = "Holi"; // let generaría error function saludar() { var saludo = "hola"; console.log(saludo+"\n"); } saludar(); console.log(saludo); // hola // Holi ```
